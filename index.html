Description: Marble Maze Game

  • Uses accelerometer and gyroscope to roll marble through a maze by tilting the device.
  • Capturing hearts increases timer by +3 seconds.
  • Game project using coremotion, spritekit, gamescene.

Challenge & Next Step

  • Our biggest challenge currently is keeping score.
  • One of the challenges we overcame was alerting when the game ended & getting gamescene and view controller to communicate with one another.
  • Implement scoring system
